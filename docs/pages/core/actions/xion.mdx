# XION Actions [Using custom XION actions with Abstract.js]

[XION](https://xion.burnt.com) works a bit differently than other Abstract-enabled chains because of their meta accounts, so this package allows for interfacing with XION via Abstract.js.

## Installation

:::code-group

```bash [npm]
npm i @abstract-money/actions-xion
```

```bash [yarn]
yarn add @abstract-money/actions-xion
```

```bash [pnpm]
pnpm add @abstract-money/actions-xion
```

```bash [bun]
bun add @abstract-money/actions-xion
```

:::
