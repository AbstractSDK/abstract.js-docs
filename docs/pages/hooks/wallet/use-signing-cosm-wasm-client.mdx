# useSigningCosmWasmClient [Hook for fetching the signing cosmwasm client]

## Import

```ts twoslash 
import { useSigningCosmWasmClient } from '@abstract-money/react'
```

## Usage

```tsx twoslash [index.tsx]
import { useSigningCosmWasmClient } from '@abstract-money/react'

function App() {
  const signingCosmWasmClient = useSigningCosmWasmClient({
    chainName: 'osmosis',
  })
}
```
