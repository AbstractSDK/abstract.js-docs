# useSenderAddress [Hook for fetching the sender address]

## Import

```ts twoslash 
import { useSenderAddress } from '@abstract-money/react'
```

## Usage

```tsx twoslash [index.tsx]
import { useSenderAddress } from '@abstract-money/react'

function App() {
  const senderAddress = useSenderAddress({
    chainName: 'osmosis',
  })
}
```
