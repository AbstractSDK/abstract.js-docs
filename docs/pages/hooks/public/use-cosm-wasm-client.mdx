# useCosmWasmClient [Hook for fetching a CosmWasmClient instance]

## Import

```ts twoslash 
import { useCosmWasmClient } from '@abstract-money/react'
```

## Usage

```tsx twoslash [index.tsx]
import { useCosmWasmClient } from '@abstract-money/react'
import { stringToAccountId } from '@abstract-money/core'

function App() {
 const cosmWasmClient = useCosmWasmClient({ chainName: 'pion' })
}
```

## Parameters

### chainName

`string | undefined`

Name of the chain to be used to fetch the query.
